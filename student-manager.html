Tổng quan dự án

Student Manager là ứng dụng Android giúp giáo viên quản lý thông tin học sinh, điểm số và theo dõi tiến độ học tập. Ứng dụng được xây dựng với kiến trúc MVVM clean architecture và tuân thủ Material Design guidelines.

Quản lý học sinh

CRUD operations đầy đủ: thêm, sửa, xóa, tìm kiếm học sinh với validation

Quản lý điểm số

Nhập điểm theo môn học, tính điểm trung bình, xếp loại tự động

Báo cáo thống kê

Xuất báo cáo PDF, biểu đồ phân tích điểm số, thống kê theo lớp

Material Design

Giao diện tuân thủ Material Design 3, responsive và accessible

Kiến trúc & Design Patterns

MVVM Architecture

Áp dụng Model-View-ViewModel pattern để tách biệt business logic và UI, giúp code dễ test và maintain.

View Layer

Activities, Fragments, Custom Views với Data Binding

ViewModel Layer

Business logic, state management với LiveData

Model Layer

Repository pattern, SQLite database, Data Access Objects

Thách thức kỹ thuật

Thách thức

Thiết kế database schema phức tạp cho quan hệ nhiều-nhiều giữa học sinh và môn học

Giải pháp

Sử dụng junction table và foreign keys, implement DAO pattern với Room database

Thách thức

Tối ưu performance cho RecyclerView với dataset lớn và complex layouts

Giải pháp

Implement ViewHolder pattern, lazy loading, và DiffUtil for efficient updates
